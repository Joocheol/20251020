프로젝트 개요

해당 분석은 과거 가격 데이터를 기반으로 포트폴리오를 전수 조사하여 수익률과 변동성 사이의 최적 조합을 찾는 과정입니다. 아래 그래프는 각 포트폴리오의 위치를 나타내며, 오른쪽 위로 갈수록 더 높은 수익과 위험을 의미합니다.

핵심 지표

데이터 출처

temp.csv에 포함된 종가 데이터를 기반으로 계산

연간화 가정

연간 거래일 252일 기준으로 수익률과 변동성을 연간화

모든 자산은 0~100% 사이의 비중, 총합 100% (롱 온리)

샘플링 간격

해석 가이드

그래프 상의 점들은 각각 가능한 포트폴리오를 나타내며, 굵은 경계선은 동일한 위험 수준에서 가장 높은 기대수익을 제공하는 포트폴리오를 연결한 것입니다. 투자자는 자신의 위험 선호도에 따라 해당 경계선 상에서 적절한 조합을 선택할 수 있습니다.
